üîå AgentConnection Class

Main class for connecting to channels and sending/receiving messages.

// Create a connection const connection = new AgentConnection({ usePubKey: false, // Optional: Use public key encryption enableWebrtcRelay: false // Optional: Enable WebRTC relay }); // Connect to a channel await connection.connect({ apiKey: 'your-api-key', // Required: Your API key baseURL: 'http://localhost:8080', // Required: API endpoint channelName: 'my-game-room', // Required: Channel name channelPassword: 'secret123', // Optional: Channel password username: 'Player1' // Optional: Display name });

üì° AgentConnection Methods

Send/receive messages and manage the connection.

  • sendMessage(data) - Send messages to all channel members
  • setCustomMessageHandler(callback) - Listen for incoming messages
  • setAgentJoinHandler(callback) - Detect when players join
  • setAgentLeaveHandler(callback) - Detect when players leave
  • storagePut(key, value) - Store persistent data
  • storageGet(key) - Retrieve stored data
  • storageDelete(key) - Remove stored data
  • disconnect() - Close the connection
  • getSessionInfo() - Get current session information
// Send a message connection.sendMessage({ type: 'player-action', action: 'jump', x: 100, y: 200 }); // Listen for messages connection.setCustomMessageHandler((message) => { console.log('Received:', message); }); // Storage API await connection.storagePut('game-state', { level: 5, score: 1000 }); const state = await connection.storageGet('game-state'); // Disconnect connection.disconnect();

üîß Additional Features

Built-in capabilities of AgentConnection:

  • Automatic WebSocket reconnection - Handles connection loss gracefully
  • Message deduplication - Built-in hash-based duplicate prevention
  • Encryption support - Optional RSA + AES encryption via MySecurity
  • FileSystem API - Upload/download files through channels
  • Event-based handlers - Join, leave, message callbacks

‚ö° WebRTCHelper Class

Manages peer-to-peer WebRTC connections for ultra-low latency communication.

// Create WebRTC helper (pass your AgentConnection) const webrtcHelper = new WebRTCHelper(connection, connection.agent); // Broadcast to all peers (P2P) webrtcHelper.broadcastDataChannel({ type: 'position', x: 150, y: 300, timestamp: Date.now() }); // Listen for P2P messages webrtcHelper.setDataChannelHandler((message) => { console.log('WebRTC message:', message); }); // Connection events webrtcHelper.setConnectionEstablishedHandler((peerId) => { console.log('Connected to:', peerId); }); webrtcHelper.setConnectionClosedHandler((peerId) => { console.log('Disconnected from:', peerId); });
